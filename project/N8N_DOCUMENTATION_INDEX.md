### Start n8n
```bash
docker-compose up -d n8n
```

### Access UI
```
http://localhost:5678
Username: admin
Password: streamlink123
```

### Import Workflow
```
Workflows ‚Üí Import from File ‚Üí Select n8n.json
```

### View Logs
```bash
docker logs -f streamlink-n8n
```

### Backup Workflows
```bash
docker exec streamlink-n8n n8n export:workflow --all
```
